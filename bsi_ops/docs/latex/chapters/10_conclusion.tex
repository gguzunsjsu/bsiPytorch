\chapter{Conclusion}
The results show that BSI can be used as a native compressed representation for LLM inference on GPUs, but only when data layout and kernel execution are treated as a single co-design problem. The stable same-bit baseline already provides meaningful compression and a usable inference path. The remaining performance gap to dense Torch is now well characterized, and the next stage of research is clearly defined: reduce arithmetic interaction cost and improve scheduler efficiency without abandoning BSI-native execution.
